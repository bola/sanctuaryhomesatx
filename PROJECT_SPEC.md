## Design & Brand Guidelines

### Color Palette
- **Cedar Charcoal** `#2B2B2B` - Primary text/background contrast
- **Warm White** `#F8F6F2` - Page background
- **Sanctuary Sand** `#E9E4DC` - Section backgrounds/cards
- **Hill Country Sage** `#6F7E6A` - Accents, links, small headings
- **Copper Accent** `#B87333` - CTA highlights, small rules

### Typography
- **Headline:** Serif (Playfair Display or Cormorant Garamond)
- **Body:** Sans-serif (Inter)

### Button Styles
- **Primary:** Copper fill with Charcoal text
- **Secondary:** Outline style
- **Tertiary:** Text only

### Logo Concept
"SCH Home" monogram - S-C-H letterforms forming a simple house silhouette

## Technical Specifications

### Performance Targets
- LCP ≤ 2.5s
- CLS ≤ 0.1
- TTI ≤ 3.5s on 4G
- WCAG 2.2 AA compliance

### Image Requirements
- Hero images ≥ 2400px width
- Gallery images ≥ 1800px
- WebP + fallback formats
- Alt-text required for all images
- Lazy-load implementation

### SEO Requirements
- URL scheme: `/projects/slug`, `/services/slug`, `/areas/slug`
- Meta descriptions: 150-160 chars
- Schema.org markup: Organization + LocalBusiness
- XML sitemap
- robots.txt

### JavaScript Features
- Image gallery/carousel with lightbox
- Portfolio filtering (by type, neighborhood, architect)
- Auto-rotating hero images (10-second timer)
- Smooth scrolling
- Form validation
- Mobile menu toggle

## Project Credit Format

**Standard Format:**
```
Architect — Interior Designer — Photography — Builder
```

**Example:**
```
JBA Architecture — WRJ Design — Amy Dang Photography — Sanctuary Custom Homes
```

**Style Rules:**
- Use official brand casing
- Use en dash (–) separators
- No trailing punctuation
- Omit unknown roles (don't use "N/A")
- Legacy builds: "Builder: Nick Chappell with Gasparini Custom Homes (legacy)"
